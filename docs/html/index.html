MTEX2Gmsh documentation

Welcome to the documentation for the MTEX2Gmsh toolbox!

This toolbox for MATLAB is intented to generate meshes from EBSD in order to perform Finite Element simulations at grain scale on crystalline materials. The following sections provide with details on using MTEX2Gmsh:
